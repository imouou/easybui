控件

控件的定义

控件：是封装简化后可嵌入页面的代码，包含 HTML 结构跟 JS 脚本，例如，下面初始化的变量名uiAccordion 称为实例，实例上有方法，通过方法可以操控实例，使用上比较灵活。

一个简单的例子，折叠菜单。

<dl id="accordion" class="bui-accordion">
  <dt class="bui-btn">A</dt>
  <dd>折叠菜单B的内容</dd>
  <dt class="bui-btn">B</dt>
  <dd>折叠菜单B的内容</dd>
</dl>

// 初始化，id参数对应html的id
var uiAccordion = bui.accordion({
  id: "#accordion",
});

// 通过点击可以展开隐藏的内容，也可以调用方法展开全部隐藏内容
uiAccordion.showAll();
